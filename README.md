# Sales Data Analysis Dashboard (Excel)

# Project Overview
This project analyzes sales data using Microsoft Excel to identify trends, patterns, and business insights.

# Tools Used
1. Microsoft Excel
2. Pivot Tables
3. Charts
4. Slicers

# Dataset Features
1. Product, Category, City
2. Quantity, Price, Total Sales
3.  Payment Method, Ratings

# Key Insights
1. Electronics category contributes the highest share of total sales
2. UPI and Credit Card are the most frequently used payment methods
3. Sales show fluctuations with peak performance in certain months
4. Laptops and smartphones are top-performing products

# Dashboard Features
1. Interactive filters (slicers)
2. Sales trends over time
3. Category-wise analysis

# Project Outcome
Converted raw data into an interactive dashboard to support business decision-making.
